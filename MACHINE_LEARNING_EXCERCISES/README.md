## Sprint 1: Nifty 50 Trading Model

**Objective**: Implement a machine learning trading model for the Nifty 50 index

**Status**: In Progress

### Directory Guide

- **data/**: Store raw data downloads (CSV, JSON, etc.)
- **features/**: Processed and engineered features ready for model training
- **models/**: Saved trained models (pickle, joblib, h5, etc.)
- **outputs/**: Results including plots, performance reports, and analysis
- **notebooks/**: Jupyter notebooks and Python scripts for data processing, training, and evaluation
